# Open Room

![Screenshot](https://github.com/ngokyuen/Open-Room/blob/main/screencapture/20220104.png)



## Build environment
1. setup node.js (https://nodejs.org/en/)
2. (command mode) go to project folder
3. run
 ```yarn```
 or ```npm install```
 
## Start Server(Dev)
run ```yarn serve```
 or ```npm run serve```
